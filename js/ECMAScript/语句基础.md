JavaScript程序的执行单位为行，也就是一行一行地执行。

语句为完成某种任务而进行的操作。

语句以分号结尾

分号也是一个执行语句，叫作**空语句**。

```js
do {
  a--;
} while ( a > 0 ); // 分号不能省略
```
